# Credit_Risk_Analysis

## Overview
The premise of the challenge was to use six different machine learning models to predict credit risk using the credit card credit dataset from LendingClub. The six models and algorithms were RandomOverSampler, SMOTE, ClusterCentroids, SMOTEENN, BalancedRandomForestClassifier, and EasyEnsembleClassifier.
